# 📈 Stock Price Prediction

A machine learning project that predicts stock prices using historical stock market data. This project demonstrates the complete workflow of a stock price prediction system, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

## 🚀 Features

- Historical stock data analysis
- Data preprocessing and cleaning
- Feature engineering
- Machine Learning based prediction
- Model performance evaluation
- Data visualization using graphs
- Predict future stock prices

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook


## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/dineshkumar200705-hue/Stock_Price_Prediction
```

Move to the project directory:

```bash
cd stock_price_prediction
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
stock_price_prediction.ipynb
```

Run all the cells to:

- Load the dataset
- Preprocess the data
- Train the machine learning model
- Evaluate model performance
- Visualize predictions

## 📊 Workflow

1. Load historical stock market data
2. Clean and preprocess the dataset
3. Perform Exploratory Data Analysis (EDA)
4. Split data into training and testing sets
5. Train the prediction model
6. Evaluate the model using performance metrics
7. Visualize actual vs predicted prices

## 📈 Sample Output

The project generates:

- Historical stock price graphs
- Training and testing results
- Predicted vs Actual price comparison
- Model accuracy metrics
